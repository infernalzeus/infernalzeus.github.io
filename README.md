# infernalzeus.github.io

Personal portfolio of **Yash Kumar (InfernalZeus)** — AI researcher and software developer.

Live at: [infernalzeus.github.io](https://infernalzeus.github.io)

## About

A single-page portfolio built with vanilla HTML, CSS, and JavaScript. No frameworks, no build step — just a static file served via GitHub Pages.

Sections:
- **Hero** — intro, links, and interactive hex-grid background
- **Projects** — selected software and research tools
- **Research** — publications and ongoing academic work
- **About** — background, skills, and accolades
- **Collaborate** — open invitation for research or engineering work

## Tech

- Fonts: [Outfit](https://fonts.google.com/specimen/Outfit) + [Orbitron](https://fonts.google.com/specimen/Orbitron) via Google Fonts
- Canvas: mouse-responsive hexagonal grid glow (hero section)
- Parallax: SVG hex/square shapes drifting in gutters
- Hosting: GitHub Pages (`main` branch, `.nojekyll`)
